## Что меняем

В `src/pages/PlayerStatistics.tsx` убираем условие `hasOut`, добавленное в прошлый раз. Result считается всегда по формуле:

```
Result = (cashout + chipOut) − (drop + chipIn)
```

для каждой строки, без проверки наличия cash out.

## Поведение после фикса

- Игрок зашёл, купил на 500k, ещё сидит → Result = **−500 000** (казино «в плюсе» на эту сумму на бумаге).
- Игрок купил на 1M, проиграл всё, ушёл без кешаута → Result = **−1 000 000** (правильный проигрыш в пользу казино).
- Игрок купил на 500k, кешаут 700k → Result = **+200 000** (выигрыш игрока).
- Если IN = 0 и OUT = 0 → Result = 0, показываем `·` (как и сейчас, для пустых строк).

Знак: положительный = игрок выиграл (красный/cms-amount-negative для казино), отрицательный = игрок проиграл (зелёный/cms-amount-positive для казино) — оставляем текущую логику окраски без изменений.
