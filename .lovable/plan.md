## Группировка слотов по часам в Breaklist

Добавить визуальные вертикальные разделители между часовыми блоками (каждые 3 слота по 20 минут), чтобы строка времени воспринималась как часовые группы.

### Что меняется
В файле `src/components/pit/BreaklistGrid.tsx` для каждой ячейки времени (как в `<thead>`, так и в `<tbody>`) определяется, является ли слот первым в часе (минуты `:00`). Для таких слотов добавляется более заметная левая граница.

### Поведение
- Слоты `XX:00` получают чуть более толстую левую границу (`border-l-2 border-border`), создавая визуальные часовые блоки по 3 ячейки.
- В тёмной теме разделитель использует `border-border` (полупрозрачный), в светлой — тот же токен, но контрастнее на белом фоне; дополнительно добавляется небольшое усиление через `border-foreground/20` для большей акцентности.
- Текущий активный столбец (`isCurrentCol`) сохраняет приоритет: его рамки `border-x-2 border-primary` остаются как есть и перекрывают часовой разделитель.
- Первый слот `18:00` тоже получает разделитель (или его можно опустить — отступ от колонки `S` уже даёт визуальную границу). Оставим разделитель для единообразия.

### Технические детали
В шапке таблицы и в каждой строке `<td>`/`<th>` для слота добавляется условный класс:
```ts
const isHourStart = slot.endsWith(":00");
// className += isHourStart && !isActive/!isCurrentCol ? " border-l-2 border-foreground/25" : ""
```
- Для `<th>`: добавить класс к существующему classname.
- Для `<td>`: добавить к существующему classname (вместе с проверкой на `isCurrentCol`, чтобы не дублировать рамки).

Никаких изменений данных, БД или поведения — только визуальная сетка часовых блоков.
